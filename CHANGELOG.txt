# Change Log
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/)
and this project adheres to [Semantic Versioning](http://semver.org/).

## [Unreleased]
### Added
- Add image processing algorithms. (Total 6 types.)
- Add hot keys to dinamically select above algorithms ('+', '-' or digit '1'..'6').
- Made it possible to reproduce the correction condition with command line.
- the unit test project "utest" added. (Powered by GoogleTest.)
- Add magnification mode to specify output image size relatively.

### Changed
- On the existing shading correction algorithm
   (currently named ImgFunc_shdc01), calcuration of binarization ROI
   improved.

### Fixed
- Slightly modified README.md and remove unused .bat file.

## [1.0.1.1] - 2024-01-16
### Changed
- Changed the command line argument specifications to be more intuitive.
  (It is now possible to specify the physical dimensions of Target drawing even in standard sizes.)
- Opencv changed to 4.8.0.
- Changed OpenCV link method to static link. 

## [1.0.0.1] - 2024-01-16
### Added
- Implemented all automatic correction functions to capture target drawing.
- OpenCV 4.6.0 was used.
- OpenCV link method is dynamic link.
